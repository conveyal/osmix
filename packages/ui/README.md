# @osmix/ui

Shared UI components for Osmix apps.

## Installation

```bash
bun add @osmix/ui
```

## Usage

```tsx
import { Button, Card, Input } from "@osmix/ui"
```

## Components

- Button
- ButtonGroup
- Card
- Collapsible
- Dialog
- HoverCard
- Input
- InputGroup
- Item
- Separator
- Spinner
- Textarea
